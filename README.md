# MNIST Handwriting Classifier

## Overview
A simple deep learning project to classify handwritten digits (0-9) using the **MNIST dataset**.  
This notebook demonstrates basic **computer vision** and **neural network** concepts in Python with TensorFlow/Keras.

## Features
- Load and visualize MNIST dataset
- Preprocess data for neural networks
- Build and train a deep learning model
- Evaluate model performance
- Make predictions on new images

## Dataset
- **Training set:** 60,000 images  
- **Test set:** 10,000 images  
- Each image is **28x28 pixels**, grayscale.
